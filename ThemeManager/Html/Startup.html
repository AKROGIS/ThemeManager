What is Theme Manager

Theme Manager simplifies access to GIS data. It provides the most benefit to occasional GIS users, or those in organizations with large data holdings. It can be used on a computer without any GIS software, but to get full access to all the features you must have ArcGIS Desktop installed (and licensed) on your computer.

The information provided below is intended to help new users get started quickly. If you have used previous versions of Theme Manager, you may want to review the section What’s New in Theme Manager 3.0 in the main help document.

Using Theme Manager

For Theme Manager to be useful, you will need a theme list (an organized collection of the GIS data available to you). Anyone can create and edit theme lists, but usually they are provided by those most familiar with the data holdings in an organization. Theme lists ready for use are displayed in the Themes panel when the Themes tab is selected on the left.

I have themes, let's go!

  • Browsing

    • Browse the treeview of themes just like a file browser. When you see a theme that is interesting, select it by clicking on it and this document will be replaced by detailed information about the theme.

  • Adding to a map

    • If you want to add a theme to a map, just double click it, or drag and drop it onto an open map document. Note that some items cannot be added to a map. The typical ones are: theme lists ( or ), categories () and sub-themes (icons vary, but text is always gray).

  • Preview

    • If you want to want see what a theme looks like before adding it to your map, first select it in the tree view, then click on the Preview tab above.

  • Searching

    • If you are looking for something in particular, enter some defining text in the search box above and press the enter key. A list of themes will be presented in the Search Results tab.

  • More Searching

    • If the number of themes returned is too many, you can refine the search by right clicking on a category (in either the Themes, Favorites, or Search Results tabs) and selecting search from the pop-up context menu. In this case you will only be searching the themes in the selected category.

  • Favorites

    • If you find a theme that may be useful again, you can add it to your list of favorites by clicking on the star () in the tool bar.

Creating Theme Lists

Creating a theme list is easy, but creating a useful theme list may require a bit more GIS knowledge than this quick start document provides. If you get stuck, see the Theme Manager help, or consult your GIS specialist.

  • Creating a new theme list

    • Select File->New Theme List from the main menu (or click on the tool bar, and then select New Theme List from the pop-up menu).

  • Adding Themes

    • Using windows file explorer to browse to any ESRI layer file (*.lyr). Select one or more files and drag and drop them onto your theme list. The layer files and any associated metadata will be added to your theme list.

    Note: Theme Manager is not limited to layer files. Any file or document can be added as a theme, however only ESRI layer files and map documents provide sub theme expansion and automatic metadata discovery. If you do not have any layer files, they will need to be created in ArcMap. Consult the ArcMap help if necessary.

  • Organizing Themes

    • First select (left mouse click) your theme list in the Themes panel, then select File->New Category from the main menu (or click ) on the tool bar, and then select New Category from the pop-up menu. Once a category is created, you can drag and drop existing items (themes or categories) into the new category folder.

    You can also create nested (sub) categories if you select an existing category instead of the theme list before invoking the New Category command.

  • Changing Names

    • To change the name of a theme list, category or theme, first click on the Properties tab above, then select the item you wish to change. Click in the top text box (labeled Theme List: Category: or Theme:) and change the name. The name will update in the tree view as soon as you click on (or tab into) another property. You can edit (or add) information in any of the text boxes that are not disabled. All fields except the name are optional.

  • Saving

    • Click on the save icon () in the tool bar, (or select File->Save or File->Save As... from the main menu). If the save icon is grayed out, then there are no new changes in your theme list that need to be saved. If your theme list has never been saved before, you will be prompted for a file name/location where it should be saved.

  • About metadata

    • Themes (as well as categories and theme lists) can have additional information about them displayed in the metadata panel. Themes will automatically discover any associated metadata, so there is nothing more that needs to be done. However If no metadata was found, you can specify it by editing the Metadata (Url/Metadata for a theme list or category) field. This field can contain a web site (http://...), a path to an XML metadata file, or a path to a data source in a geodatabase that has embeded metadata. If the metadata is an XML file you can use the browse (...) button to find it. Themes will extract additional information (publication date, description, ...) from metadata to help with searching. IF your theme has no metadata you can add your own information in these fields, or you can use the ArcCatalog to create metadata for your theme. Consult the ArcCatalog help if necessary.
